Вариант 110: Конкатенация строк
Вам дан односвязный линейный список, элементами которого являются строки произвольной длины.
Объедините эти строки в одну. Предложите наиболее эффективную реализацию.

Почему предложенная реализация наиболее эффективна?
1. Один раз выделяется память под итоговую строку
2. Минимальное количество вызовов malloc
3. Использование strcat в финальном проходе — безопасно и логично
4. Временная сложность: O(N)
